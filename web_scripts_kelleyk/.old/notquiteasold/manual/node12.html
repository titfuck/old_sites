Voicemail Hardware

Voicemail hardware consists of the Voicemail Computer, as well as some custom hardware that is installed above the Patch Box. You will rarely have to worry about the hardware above the patch box. The key point is that this hardware must be plugged into the Voicemail Computer with an RJ-45 cable.
