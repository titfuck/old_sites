tdc-library-2.mit.edu printer/dhcp server

tdc-library-2.mit.edu acts as the print server for the HP-4050, and as such provides printing through UNIX LPR queues, Appletalk, or Windows Networking. The printer must always be connected to this computer, which should always be connected to the net.

Additionally, this computer serves as the house's DHCP server, which provides dynamic networking information to computers that do not have static IP addresses. 1 More on this later.
