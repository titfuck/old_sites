The Physical Connections

First, the following sets of RJ-45 cables must be installed:
  • From the computer's ethernet card to a network drop in the wall. For example's purpose, let's assume the computer is plugged into 3-3A.
  • In the network closet, from 3-3A into a hub.
  • From the hub to the MIT drop, or to another hub that is somehow connected to the MIT drop.

When this set of connections has been made, the ``link'' light on the computer's network card should light up. This light is usually green.
