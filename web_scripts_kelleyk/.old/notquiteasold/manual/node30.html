DHCP

DHCP is by far the easiest way to get someone on the network. Currently the network range from 18.237.0.75 to 18.237.0.99 is reserved for DHCP addresses. If a computer is set to use DHCP, it will automatically configure all of its networking. There is no need to assign IP addresses on a per-computer basis when using DHCP.
