TDC.MIT.EDU

The TDC Website ``Network / Phone'' section provides various features for regular users as well as for the netgeek.

Users can:

  • request phone/ethernet ports
  • find out their IP address
  • get a list of phone extentions

As the netgeek, you can:

  • view a list of requested phone/ethernet ports, and quickly make the changes in the Network Closet. The website will also email you requests that have been made, so you know when you have work to do in the closet.
  • edit the list of IP addresses in use 2
  • add phone extentions to the web listing 3
