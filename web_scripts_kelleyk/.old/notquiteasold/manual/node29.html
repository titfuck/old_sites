Getting new static IP addresses

First check the tdc.mit.edu website to see if there is an available IP address that is either registered to ``Test User'' or that is registered to someone that no longer lives at TDC. If you try to ping the computer and get no response, then you can re-use that address. If there are no such addresses, you will need to create a new IP address.

On Athena, use moira to create a new IP address. Under ``(4) machine menu,'' choose ``(4) add,'' and fill out the requested information. ``Network Assignment'' should be filled out as ``tdc.''
