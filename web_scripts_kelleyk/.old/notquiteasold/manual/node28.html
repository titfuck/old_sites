Networking Numbers

If you are giving a computer a static IP address, you will need to specify the following network parameters:
  • IP Address - Distribute an unused IP address from tdc.mit.edu, or create a new IP address with moira
  • Gateway - 18.237.0.1
  • Netmask - 255.255.255.0
  • DNS (or Nameserver) - 18.70.0.160, 18.72.0.3

It is only necessary for someone to have a static IP address if they are planning to run some sort of server that requires they know their IP address or hostname. If they do not have this requirement (which summer boarders and incoming freshmen will not have) then it is easier for them to use DHCP.
